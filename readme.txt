This Theme works with WP as Backend and Angualar.js in Frontend using WP Rest API Plugin


== Installation ==

Requirements:

1.  Activat Permalinks by Slug
2.  Install the following Plugins

<div>
  <ul>
    <li>
      WP Rest API (v2) by <a href="http://v2.wp-api.org/" target="_blank">WP REST API Team</a>
    </li>
    <li>
      WP Rest API Menus by <a href="https://github.com/unfulvio" target="_blank">Fulvio Notarstefano</a>
    </li>
    <li>
      WP Rest API Sidebars by <a href="https://github.com/martin-pettersson" target="_blank">Martin Pettersson</a>
    </li>
  </ul>
</div>

 

3. In your admin panel, go to Appearance > Themes and click the Add New button.
4. Click Upload and Choose File, then select the theme's .zip file. Click Install Now.
5. Click Activate to use your new theme right away.

 

You are able to customize the Frontpage by editing the main.html file in THEME-folder/partials/

These are the build-in directives:

*   <content-search-posts></content-search-posts>
*   <apt-banner-image wp-src="wp-content/uploads/2016/05/background1.jpg"></apt-banner-image>
*   <single-page page-slug="example-page"></single-page>
*   <apt-image wp-header="Header 2" wp-Text="A description Text" wp-position="left"></apt-image>
*   <single-post post-slug="'hallo-world'"></single-post>
*   <apt-image wp-header="Header 3" wp-Text="A description Text"></apt-image>
*   <posts-by-category category-slug="'News'"></posts-by-category>
